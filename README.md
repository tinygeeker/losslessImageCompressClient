# 图片无损压缩客户端 🖼️

一个基于PyQt5和Pillow的图片无损压缩客户端，支持多图片批量压缩、实时显示压缩进度、单张下载或批量打包下载等功能。

## 功能特点

### 🎯 核心功能
- **🐣 多图片批量压缩**：支持同时添加和压缩多张图片
- **⏳ 实时压缩进度**：显示实时压缩进度和剩余时间
- **📥 多种下载方式**：单张图片下载或批量打包下载
- **📊 压缩比例统计**：显示每张图片的压缩比例和最终大小
- **🖼️ 图片格式支持**：支持JPEG、PNG、WEBP等多种图片格式

### 🎨 界面设计
- **🌈 现代化界面**：简洁美观的用户界面
- **🖼️ 图片缩略图**：显示每张图片的缩略图
- **📋 详细信息**：显示图片格式、大小、剩余时间等详细信息
- **📈 进度条显示**：实时显示压缩进度，完成时显示100%
- **📱 响应式布局**：适配不同屏幕尺寸

### 🔧 技术特性
- **⚡ 多线程处理**：使用多线程优化压缩速度
- **🔒 无损压缩**：保持图片质量的同时减小文件大小
- **🛡️ 错误处理**：完善的错误处理机制
- **👨‍💻 用户友好**：直观的操作流程，易于使用

### 📖 附加功能
- **ℹ️ 关于页面**：显示软件信息和版本
- **🌐 开发者主页**：链接到开发者的GitHub页面
- **❤️ 打赏功能**：支持通过扫描二维码进行打赏

## 系统要求 🖥️

- Python 3.13.2
- PyQt5
- Pillow

## 安装依赖 📦

```bash
pip install pyqt5 pillow
```

## 运行应用 🚀

```bash
python main.py
```

## 使用说明 📝

1. **📁 添加图片**：点击"添加图片"按钮选择要压缩的图片
2. **⚡ 自动压缩**：系统会自动开始压缩，实时显示进度和剩余时间
3. **📊 查看结果**：压缩完成后，查看压缩比例和最终大小
4. **📥 下载图片**：点击"下载"按钮保存单张图片，或点击"批量下载"按钮打包下载
5. **❤️ 打赏支持**：在"关于"菜单中点击"打赏"选项，扫描二维码支持开发者
